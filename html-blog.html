Pengetahuan Singkat Tentang HTML

HTML merupakan singkatan dari HyperText Markup Language yang dimana merupakan sebuah bahasa yang digunakan untuk membuat halaman website. HTML berperan penting dalam menentukan struktur sebagai landasan awal dari konten dan tampilan sebuah website. Membuat website tidak hanya menggunakan html saja, terdapat bahasa lain yang dapat digunakan untuk mempercantik dari sebuah website yang kita buat, yaitu CSS dan JavaScript yang dimana nanti kita bahas di artikel berikutnya.

Sebelum membuat website diperlukan sebuah text editor yang dapat membantu kita dalam mendevelopment website, software tersebut diantara lain yaitu :

  1. Notepad++
  2. Sublime Text
  3. Visual Code Studio
  4. Atom
  5. Dan sebagainya.
