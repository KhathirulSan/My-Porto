Pengetahuan Singkat Tentang JavaScript

JavaScript merupakan bahasa pemrograman yang tidak dapat kita lewati pada saat mendevelopment sebuah website. Karena pada dasarnya JavaScript adalah sebuah bahasa pemrograman yang dari awal dirancang untuk dapat berjalan diatas sebuah browser, akan tetapi semakin melesatnya dunia teknologi, penggunaan JavaScript tidak hanya untuk website, bahasa tersebut dapat digunakan untuk mendevelopment Game, Iot, Dekstop dan sebagainya.

Alasan terbaik kenapa programer mulai dari tahap pemula sampai tingkat expert adalah bahasa yang dapat membantu kita dalam mengembangkan sebuah aplikasi web dekstop maupun mobile, dan membuatnya menjadi interaktif. Selain itu bahasa pemrograman JavaScript saat ini memiliki banyak sekali Framework dan library yang dapat kita gunakan untuk mempercepat sebuah proses development seperti, AngularJS, jQuery, dan React JS.
