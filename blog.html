Pengetahuan dasar tentang HTML

Hypertext Markup Language (HTML) adalah bahasa pemrograman standar dalam pembuatan halaman web. Fungsi utama HTML adalah untuk membangun struktur sebuah halaman web dengan berbagai elemen di dalamnya.

Pengetahuan dasar tentang CSS

CSS adalah salah satu styling language (bahasa desain), bagian dari markup language yang dapat “mewarnai” atau mendesain suatu halaman website. CSS sendiri adalah singkatan dari Cascading Style Sheet.

Pengetahuan dasar tentang JavaScript

JavaScript adalah bahasa skrip yang digunakan untuk membuat konten halaman web dinamis, berfungsi untuk membuat elemen yang mampu meningkatkan interaksi pengunjung seperti menu drop-down, animasi, dan sebagainya.
